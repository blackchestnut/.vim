h1. Usage

Clone this repo into your home directory either as .vim (linux) or vimfiles (windows).

Then cd into the repo and run this to get all plugins installed:

<pre><code>./update_bundles
</code></pre>

Put this in your ~/.vimrc along with your personal hacks:

for linux
<pre><code>source ~/.vim/vimrc
source ~/.vim/gvimrc
</code></pre>

for windows
<pre><code>source ~/vimfiles/vimrc
source ~/vimfiles/gvimrc  
</code></pre>
